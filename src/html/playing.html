Running a battle


Click either the Run or Step buttons to start a game. The Add Ruler dialog will be displayed which allows you to select which rulers you want to do battle.
You can also Import any rulers that you want in the game (if they haven't already been imported).
Select what rulers you want in the game.
You may select any ruler multiple times if you want your ruler to battle against itself.

Click OK to start the battle.

If you initially clicked the Run button then the battle will immediately commence. You may click the pause button at any time to pause the game. Click the Run button again to resume the battle or click the Step button to watch the battle one turn at a time.

If you initially clicked the Step button then you need to click the Step button to step the battle one turn at a time, or you can click the Run button to run the game.

You may click the Stop button at any time to stop the game (after which it cannot be resumed).
